# PRD：聆海智能 · 社媒运营小龙虾平台 v1.0

## 1. 产品概述

### 1.1 产品名称
聆海智能 · 社媒运营小龙虾平台（LinkSea AI · Social Media Lobster Platform）

### 1.2 产品定位
为聆海智能运营团队提供的**社交媒体内容全生命周期管理平台**。连接后端 AI Agent 系统（OpenClaw），让运营人员可以在 Web 端查看、管理和协作所有社媒运营活动和报价项目。

### 1.3 目标用户
- 聆海智能内部运营人员（3-5 人）
- 内容策划、文案撰写、视觉设计角色
- 未来可能扩展为 SaaS 服务面向外部客户

### 1.4 核心价值
- **一站式查看**：所有 Campaign 和 Project 的内容、素材、配图集中管理
- **可视化协作**：替代飞书群和文件夹翻找，让团队快速了解项目状态
- **内容预览**：在发布前预览各平台的图文效果

## 3. 页面详细设计

### 3.1 首页 Dashboard

#### 3.1.1 顶部导航栏（固定）
- **左侧**：品牌 Logo + 产品名"聆海智能 · 社媒运营小龙虾平台"
- **右侧**：用户头像（v2 再做登录体系）
- **导航标签**：`运营活动`（v1 只做这一个）

#### 3.1.2 数据统计区
4 个统计卡片，横排排列：

| 卡片 | 数据 | 说明 |
|------|------|------|
| 运营活动总数 | 数字 | 当前 Campaigns 总数 |
| 进行中 | 数字 | 有 master-copy.md 但未完成所有平台分发的 |
| 已完成 | 数字 | 所有目标平台都有 content.md 和配图的 |
| 本周新增 | 数字 | 本周创建的 Campaign |

#### 3.1.3 项目列表区

**视图模式**：卡片视图（默认）/ 列表视图（可切换）

**卡片设计规范**：
- 尺寸：宽度自适应，最小 320px，3 列网格
- 封面图：取项目中的第一张图片（优先 cover 图），高度 180px，`object-fit: cover`
- 右上角标签：显示内容主题（产品展示/行业案例/品牌动态/技术幕后）
- 项目编号：小字红色，如 `CFQjiHX`
- 项目标题：从 brief.md 的标题提取
- 底部元信息：文件数、图片数、最后更新时间
- 状态指示：
  - 🟡 策划中 — 只有 brief.md
  - 🔵 撰写中 — 有 master-copy.md
  - 🟢 已完成 — 有平台分发稿和配图
  - 🔴 待审核 — 有内容但未标记完成

**排序**：默认按最后更新时间倒序

**搜索/筛选**（顶部工具栏）：
- 搜索框：按项目名称/编号搜索
- 筛选器：按状态、按主题分类

---

### 3.2 Campaign 详情页

点击卡片进入，页面结构为**左右布局**：

#### 3.2.1 左侧：项目信息面板（固定宽度 320px）

**项目卡片**：
- 项目编号
- 项目标题
- 内容主题标签
- 创建时间
- 最后更新时间
- 状态标签

**阶段进度**（垂直时间线）：
```
● 策划完成 — 2026-03-25
   brief.md ✅
● 母稿完成 — 2026-03-25
   master-copy.md ✅
● LinkedIn — 2026-03-25
   content.md ✅ | 配图 3张 ✅
○ 小红书 — 待处理
   content.md ❌ | 配图 ❌
○ 公众号 — 待处理
```

**快捷操作**：
- 下载全部文件（打包 zip）
- 上传素材

#### 3.2.2 右侧：内容区域（Tab 切换）

**Tab 1：概览 (Overview)**

Brief 内容的卡片化展示：
- 核心卖点（3 个要点，分别一张小卡片）
- 深度事实描述（可展开/收起）
- 视觉资产缩略图网格（点击放大）

**Tab 2：素材库 (Assets)**

- 网格展示所有素材图片（缩略图 + 文件名 + 文件大小）
- 支持点击放大预览（Lightbox 模式）
- 支持拖拽上传新素材
- 图片下方显示语义化描述（如果有 _description.md 文件）

**Tab 3：母稿 (Master Copy)**

- 母稿内容的富文本渲染（Markdown → HTML）
- 配图标记位置高亮显示，悬浮显示对应素材图缩略图
- 版本历史（如果有 master-copy_v1.md 等旧版本）

**Tab 4：平台分发 (Distribution)**

- 顶部：平台标签切换（LinkedIn | 小红书 | 公众号 | Facebook）
- 每个平台的内容展示：

**LinkedIn 视图**：
- 模拟 LinkedIn 帖子卡片样式
- 左侧头像 + 公司名
- 正文内容
- 配图展示
- Hashtags

**小红书视图**：
- 模拟小红书笔记样式
- 封面图大图
- 左右滑动浏览多张内容图（轮播/横向滚动）
- 标题（大字）
- 正文
- 标签云

**公众号视图**：
- 模拟公众号文章样式
- 封面图
- 标题 + 摘要
- 正文（长文排版）

**Facebook 视图**：
- 模拟 Facebook 帖子样式
- 配图
- 正文
- Hashtags

**Tab 5：文件管理 (Files)**

- 树形结构展示项目所有文件
- 支持按文件类型筛选（图片/文档/JSON）
- 每个文件支持：预览、下载、删除（v2）

## 4. 交互规范

### 4.1 图片预览
- 点击任意图片 → Lightbox 模式全屏预览
- 支持左右切换同一组图片
- 支持键盘 ESC 关闭、← → 切换
- 底部显示文件名和尺寸

### 4.2 文件上传
- 拖拽上传：支持拖拽文件到素材区域
- 点击上传：点击上传区域弹出文件选择
- 上传进度：显示进度条
- 上传完成：自动刷新素材列表

### 4.3 Markdown 渲染
- 标题层级样式
- 表格样式（带边框、斑马纹）
- 代码块高亮
- 图片内联显示

### 4.4 响应式
- 桌面端：三栏卡片网格
- 平板端：两栏
- 最小宽度：1024px（内部工具，不需要移动端适配）

## 5. 视觉设计规范

### 5.1 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#EF4444` | 品牌红（小龙虾红）|
| 背景主色 | `#0A0E17` | 深色背景 |
| 卡片背景 | `#111827` | 略浅于背景 |
| 边框 | `#1E2536` | 低对比度分割 |
| 正文 | `#E0E0E0` | 主文字 |
| 辅助文字 | `#64748B` | 次要信息 |
| 成功 | `#22C55E` | 已完成状态 |
| 警告 | `#F59E0B` | 进行中状态 |
| 信息 | `#3B82F6` | 链接/可操作 |

### 5.2 字体
- 中文：PingFang SC / 思源黑体
- 英文：Inter / SF Pro
- 代码：JetBrains Mono

### 5.3 圆角
- 卡片：12px
- 按钮：8px
- 标签：20px（胶囊型）
- 输入框：8px

### 5.4 间距系统
- 页面边距：24px
- 卡片间距：16px
- 卡片内边距：16-20px
- 元素间距：8px / 12px / 16px / 24px

### 5.5 品牌元素
- Logo：🦞 小龙虾 emoji + "聆海智能" 文字
- 空状态插图：小龙虾卡通形象（可让设计师发挥）

## 6. 技术约束

### 6.1 后端
- Python HTTP Server（已有基础代码）
- 数据来源：本地文件系统，无数据库
- API 接口：RESTful JSON

### 6.2 前端
- 纯前端 HTML/CSS/JS，不使用框架（保持轻量）
- 或可选用 Vue 3 CDN 版本提升开发效率
- 所有资源内嵌在单个 HTML 文件中（方便部署）

### 6.3 部署
- 运行在 Mac Mini 上（内网访问）
- 端口：8903
- 内网地址：http://172.16.7.22:8903
