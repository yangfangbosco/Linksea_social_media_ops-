# CPzZAny / LinkedIn 配图建议书

## 项目判断
这是一条典型的 **B2B 案例型 LinkedIn 内容**。
它要传达的不是“炫技”，而是：**腾讯滨海展厅如何把原本分散的 AI 能力，收束为一个统一、自然、可持续运营的交互入口。**

因此，LinkedIn 配图策略不应走海报化、营销感过强的路线，而应坚持：

- **企业级可信感**：真实场景、真实空间、真实落地
- **系统级表达**：强调“统一入口”而不是单点功能
- **克制的科技感**：可以有未来感，但不能像消费级炫光海报
- **阅读友好**：适配 LinkedIn 竖向滑读节奏，首图必须一眼建立案例级别

---

## 总体配图策略
建议本条 LinkedIn 内容采用 **1 张封面主视觉 + 3 张文内配图** 的结构。

原因：
- LinkedIn 用户更适合看 **少而准** 的案例配图，不适合过多碎图打散节奏
- 当前素材强项在于 **空间氛围、品牌场景、展厅落地感**，适合做“案例叙事型”配图
- 文案的核心逻辑是从 **问题 → 统一入口 → 自然交互 → 可持续运营** 递进，因此图片也要跟着这条叙事走

推荐顺序：
1. **封面图：建立案例级别与品牌信任**
2. **入口图：建立统一交互入口的空间印象**
3. **讲解/体验图：体现沉浸式、多媒体联动与自然交流氛围**
4. **运营图：体现空间可延展、可持续更新的系统感**

### 2. 入口场景图
- **建议图片**：`assets/image_02.jpg`
- **对应段落**：统一交互入口 / 参观者第一接触点
- **为什么选它**：
  - 画面有明显的装置中心与发散式光线，具备“入口装置 / 交互起点”视觉暗示

### 3. 讲解 / 体验场景图
- **建议图片**：`assets/image_04.jpg`

- **为什么选它**：
  - 长廊式动线非常适合表达“参观过程中连续发生的体验”

- **叙事价值**：
  - 它不是最炸裂的一张，但非常适合解释“自然交互如何嵌入参观路径”
  - 能帮助读者理解：体验升级发生在空间流动中，而不只是在一个点位上

### 4. 运营 / 可持续迭代场景图
- **建议图片**：`assets/image_05.jpg`

  - 画面更完整地呈现了展厅界面、品牌墙、多媒体屏幕与空间深度
  - 反光地面与延展型曲线墙面强化了“系统、持续、可扩展”的感受
  - 比 image_04 更适合承接“长期运营”这个更宏观的段落
- **使用方式**：
  - 适合放在文章后段，作为“这不是一次性项目，而是可持续演进空间”的视觉收束

## 视觉执行原则
- **色调**：保留原图蓝色系与企业科技感，不额外加重饱和度
- **文字压图**：尽量少；若必须压字，只压封面图，且避开 logo 与主装置
- **不要做的事**：
  - 不要做成招聘海报风
  - 不要加过多图标、箭头、参数说明
  - 不要使用夸张霓虹特效覆盖真实场景
  - 不要把图片裁得只剩局部光效，丢失“真实落地案例”证据感
