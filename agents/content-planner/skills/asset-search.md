# Skill: asset-search — 素材搜索

## 用途
根据用户描述，搜索并整理符合要求的图片或文案素材。

## 输入
- Campaign ID
- 用户的搜索描述（如"高清商场室内图"、"游乐园场景图"、"竞品产品介绍"）

## 处理逻辑
1. **分析用户需求**：判断要找图片素材还是文案素材
2. **搜索**：使用网络搜索相关内容
3. **筛选**：从搜索结果中筛选高质量、相关度高的内容
4. **整理**：
   - 图片素材：列出建议下载的高清图片 URL，命名为有意义的文件名（如 `mall-interior-01.jpg`）
   - 文案素材：将找到的文案内容整理成可保存为 `assets/reference-[主题].md` 的格式
5. **记录**：在输出末尾给出"搜索了什么、找到了什么、建议保存哪些文件"的清单

## 保存路径
所有素材应归档到：`Campaigns/CXXXXXX/assets/`

## 注意事项
- 图片只推荐高清、无水印、可商用的
- 文案素材必须注明来源
- 文件命名要有意义，不要用随机字符
- 每次搜索建议 3-5 个素材即可，不要太多
